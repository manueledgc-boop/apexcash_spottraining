# VISIÓN DEL PROYECTO

ApexCash Tournament Trainer nace con una misión completamente distinta a la mayoría de entrenadores de póker existentes.

Su objetivo NO es enseñar teoría.

Su objetivo NO es enseñar rangos.

Su objetivo NO es enseñar ICM mediante vídeos.

Su objetivo consiste en desarrollar la capacidad del jugador para tomar decisiones correctas durante un torneo completo.

El usuario no estudia conceptos.

El usuario vive un torneo.

Cada spot representa una situación real.

Cada decisión tiene un contexto diferente.

Cada contexto obliga al jugador a adaptarse.

Cuando el usuario termine Tournament Trainer Standard deberá sentir que ha vivido el equivalente a decenas de torneos reales antes de sentarse a jugar uno con dinero.

Ese será el verdadero valor del proyecto.

No del estudio pasivo.

---

# OBJETIVO PEDAGÓGICO

No queremos que el usuario memorice rangos.

Queremos que aprenda a interpretar correctamente el contexto de un torneo.

Un mismo AJo puede ser:

• Fold

• Raise

• Call

• All-in

Dependiendo de:

* la etapa del torneo;
* su stack;
* el stack efectivo;
* la posición;
* la acción previa;
* la presión ICM.

Tournament Trainer enseñará exactamente esa capacidad de adaptación.

# EL CONTEXTO ES EL REY

Cada spot será construido a partir de un contexto.

El contexto define la decisión.

La mano solamente forma parte del contexto.

Cada spot incluirá:

• Etapa del torneo

• Field inicial

• Jugadores restantes

• Puestos premiados

• Mesa de 9 jugadores

• Hero Position

• Hero Stack

• Effective Stack

• Acción previa

• Calle

• Board (si existe)

• Pot (si existe)

• Mano Hero

Todo ello construye la situación.

# SISTEMA XP

El XP NO representa puntos.

Representa EXPERIENCIA SIMULADA.

Cuanto mayor sea el XP, mayor será la experiencia adquirida jugando torneos virtuales.

El recorrido será mucho más largo que el Cash Trainer.

Objetivo aproximado:

60.000 - 90.000 XP

Esto refleja que dominar torneos requiere mucha más práctica que dominar cash.

# TOURNAMENT SIMULATION

La Tournament Simulation sustituye al examen tradicional.

No se desbloquea mediante XP.

Se desbloquea únicamente cuando todas las etapas anteriores están completadas.

Características:

120-150 decisiones.

Sin feedback.

Sin pistas.

Sin porcentaje durante la simulación.

El usuario únicamente verá:

Tournament Simulation

Spot

63 / 150

Y continuará jugando.

Como un torneo real.
